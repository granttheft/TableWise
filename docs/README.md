# TableWise Dokümantasyon

Bu klasör proje dokümantasyonunu içerir.

## İçerik

- **Tablewise_Urun_Dokumani_v1.1.docx** - Ana ürün dokümantasyonu (buraya eklenecek)

## Dokümantasyon Yapısı

### API Dokümantasyonu
API dokümantasyonu Swagger/OpenAPI üzerinden otomatik olarak oluşturulur.
- Geliştirme: `http://localhost:5000/swagger`
- Production: `https://app.tablewise.com.tr/swagger` (sadece yetkili kullanıcılar)

### Mimari Kararlar
Mimari kararlar (Architecture Decision Records - ADR) için ayrı bir klasör oluşturulabilir.

### Kullanım Kılavuzları
Son kullanıcı ve admin panel kullanım kılavuzları için ayrı dokümanlar eklenecek.

## Notlar
- Tüm teknik dokümantasyon İngilizce yazılacak
- Kullanıcı dokümantasyonu Türkçe olacak
- API değişiklikleri changelog'da takip edilecek
